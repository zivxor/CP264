bool graph_am_add_edge ( graph_am * source,
const graph_am_edge * edge )

Adds an edge to source. Fails if vertex values invalid or edge already in source. edges containing row < 0 or row >= size are ignored; edges containing col < 0 or col >= size are ignored; edges containing row == col are a loop and are given a value of 2; All other edges are given a value of 1 in [row][col] and [col][row].

Parameters

Returns
- true if the edge was added, false otherwise

◆ graph_am_create()

graph_am * graph_am_create ( int size,
const graph_am_edge * edges,
int count )

Creates a graph from an array of edges. edges containing row < 0 or row >= size are ignored; edges containing col < 0 or col >= size are ignored; edges containing row == col are a loop and is given a value of 2; All other edges are given a value of 1 in [row][col] and [col][row]. The resulting matrix is symmetrical.

Parameters
size- height/width of the graph matrix
edges- array of edges
count- number of values in edges
Returns
- a new graph
